HTML 解析相关工具和库，可用于 Office 文档处理和网页爬虫场景。

## 框架

- AngleSharp：HTML 解析库（读取/修改元素、CSS 选择器）
- HtmlAgilityPack：HTML 解析库（XPath 解析、字符串转 DOM）
- PuppeteerSharp：浏览器自动化工具
