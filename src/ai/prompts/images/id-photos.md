## 领导干部风格

```
参考上传照片中的人物，保持人物真实身份、五官比例、脸型、年龄感、发型和自然特征不变。可以适度优化面部轮廓，使脸部线条更清晰、自然、上镜；平滑皮肤质感，减少瑕疵、暗沉和肤色不均，但不要过度磨皮，保留真实皮肤细节和自然光影。将人物生成中国领导干部标准证件照 / 组织任免公示照的风格。正面头肩半身构图，竖版3:4，人物居中，头部和肩膀占画面主要区域，裁切到胸口附近。头如果歪了调整为正面的，人物肩膀平正，姿态端正，可以稍微倾斜一些身体，直视镜头，表情庄重自然，轻微克制微笑，不露齿。穿深色保守正式西装、白色衬衫、正式领带，领带居中，西装传统版型。背景为传统官方蓝色渐变证件照背景，上方中蓝，下方浅蓝，干净无场景。光线为传统影楼证件照正面柔光，脸部光线均匀，阴影很轻，皮肤自然轻微修图，保留真实纹理，五官清晰。整体气质正式、稳重、朴素、端庄、可信赖，像组织部干部履历头像、机关领导干部公示照片、新华社式官方证件照，真实摄影质感。

负面词： 不要横版，不要16:9，不要企业宣传照，不要领英头像，不要现代商务大片，不要电影感，不要强烈光影，不要深蓝科技背景，不要办公室背景，不要户外背景，不要复杂背景，不要侧脸，不要侧身，不要低头，不要仰拍，不要大笑，不要露齿笑，不要网红感，不要偶像感，不要时尚写真，不要潮流西装，不要过度磨皮，不要塑料皮肤，不要夸张美颜，不要背景文字，不要红色边框，不要公章，不要组织名称。
```

## 领英证件照

```
请基于我上传的人像照片生成一张正式商务档案头像海报。保留人物真实身份特征、五官比例、脸型基础和整体气质，不要改变人物身份。 可以适度优化面部轮廓，使脸部线条更清晰、自然、上镜；平滑皮肤质感，减少瑕疵、暗沉和肤色不均，但不要过度磨皮，保留真实皮肤细节和自然光影。 请根据人物脸型设计一款匹配的商务发型。发型应干净利落、成熟专业、适合正式场合，发丝自然、有层次，不夸张。请为人物搭配正式商务穿搭，例如深色西装外套、白色或浅色衬衫，可搭配领带。整体造型应高级、简洁、专业、可信赖。 画面为正面半身肖像，人物居中，直视镜头，表情自然自信。背景为纯白色素色背景，光线柔和均匀，类似专业证件照、企业头像摄影或大学官网个人档案照。整体风格干净、正式、商务、高清写实摄影质感，方形比例 1:1。 画面下方预留简洁信息栏，并添加现代简洁排版文字： 姓名：[Harry] 职位：[产品经理] 部门：[产品管理部] 重要文字规则： 如果用户输入的姓名、身份、职位、专业、部门、机构或其他需要显示在图片上的文字是中文或其他语言，请先将这些内容翻译成自然、准确、正式的英文，再放入图片中。除非用户明确要求保留中文，否则图片中不要显示中文文字。 排版要求： 第一行姓名使用深蓝色粗体大字号英文文字；第二行和第三行使用黑色常规字号英文文字。整体排版参考大学官网个人档案、学术会议人物介绍卡片或专业商务头像海报风格。
```
