## 游戏海报

```
生成高完成度史诗感艺术海报，双重曝光构图，米白色背景，游戏：The Witcher 3，杰洛特的侧脸大剪影占据主体，剪影内部融合希里、叶奈法、特莉丝、凯尔莫罕城堡、狼派徽章等元素。整体以灰蓝、土褐、为主，压抑、决绝、宿命感极强，大量留白，印刷颗粒质感，像正式院线动画电影海报，竖版。
```

## 动漫海报

```
乐观史诗感艺术海报，竖版构图，双重曝光手法。背景为明亮的米白色肌理底纹，大面积留白。画面中心是哆啦A梦的正脸大特写，直面镜头，神情自信温暖，嘴角上扬，散发着坚定的守护感。 蓝色剪影内部融合着充满希望的元素：散发着金光的任意门半开着，绚烂的樱花雨飘落，竹蜻蜓划出闪亮的轨迹，一道七彩彩虹贯穿其中。 整体色调从之前的压抑转为明快：保留经典的钴蓝色，并融入暖阳黄、樱花粉与春意绿。氛围充满纯粹的喜悦、无限的可能与治愈感。35mm胶片颗粒质感，高光灿烂，对比度适中，宛如一部宏大冒险故事的开篇。杰作，8k分辨率，正式院线电影海报质感。
```
