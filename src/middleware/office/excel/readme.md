Excel 电子表格处理相关工具和库。

## 框架对比

| 特性                | NPOI              | DocumentFormat.OpenXml | EPPlus           | MiniExcel        |
| ------------------- | ----------------- | ---------------------- | ---------------- | ---------------- |
| **易用性**          | 中等偏下          | 中等                   | 高               | 高               |
| **上手难度**        | 较大              | 中等偏高               | 中等             | 低               |
| **支持低版本Excel** | 支持（XLS, XLSX） | 不支持（仅XLSX）       | 不支持（仅XLSX） | 不支持（仅XLSX） |
| **大文件内存占用**  | 较高              | 较低                   | 较高             | 较低             |
