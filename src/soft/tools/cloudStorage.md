## Webd

一款极其轻量级的自建网盘工具，具有精巧简洁的设计（服务端程序仅为60～90KB），而且完全免费，没有任何外部依赖。它不仅操作简单，还能以高速度进行文件传输，资源占用也非常低。

支持使用网页端即可上传、下载文件，支持用户权限管理，密码下载等功能。



Webd具备跨平台支持，可在Windows、Linux，甚至OpenWRT（路由器）、Android等多种设备上运行。在运行之前，你只需编辑它的配置文件webd.conf，配置文件包括需要共享的文件夹路径（即网盘的根目录）、端口号，以及用户名、密码和权限等设置。启动服务后，你就可以通过浏览器轻松访问并使用该网盘功能了。这一切操作简单明了，让你更轻松地管理你的文件。



使用方法如下：

1. 首先，访问下载页面并根据你的操作系统选择相应的下载文件，通常是一个压缩包。例如，如果你使用的是Windows系统，你可以选择下载名为"webd-20220512-win32.7z"的文件。

2. 下载完成后，使用压缩软件打开下载的压缩文件。

3. 在压缩文件中找到名为"webd.conf"的文本文档，双击打开它。

4. 在打开的"webd.conf"文本文档中，你可以看到各种配置选项。根据你的需求，配置需要的信息，如文件夹路径、端口号、用户名、密码以及权限等等。确保你按照要求正确填写了这些配置项。

5. 配置完成后，保存并关闭"webd.conf"文本文档。

6. 启动Webd服务。具体启动方法可能因操作系统而异，请根据所提供的文档或指南进行操作。

7. 一旦Webd服务启动成功，你可以通过浏览器访问它，以便开始使用自建网盘功能。

```

❤ 注：配置文件里以井号开头的行都是"注释"，并不生效。
❤ 如需让配置生效，需要删除配置前的"#"号。
Webd.Root：指定网盘文件的路径；
Webd.Listen 监听端口或特定的地址，支持多个，可配置成 [::]:9212 来同时监听 IPv6 和 IPv4（不会设置默认也行）；
Webd.Hide：隐藏托盘图标, 无参数，该项仅支持 Windows（不用管）
Webd.User：设置用户的权限、用户名和密码，支持两个用户，但使用同一目录，如：Webd.User rlumS user1 pass1表示设置user1的密码为pass1。
有：
① r（访问文件）；
② l（获取文件列表）；
③ u（上传文件）；
④ m（删除移动重命名文件）；
⑤ S（显示隐藏文件）等权限，可任意配置单个或多个权限。
Webd.Guest：设置无需登录的访客权限，参考上面的权限组合；设置成 0 表示禁用访客。
Webd.Browser：用于指定自定义的浏览器路径，如："D:\Microsoft Edge\edge.exe"
配置完成后，点击"webd.exe"运行，可以在底部菜单查看到托盘，点击右键调出菜单，点击"Open by Browser"就可以在浏览器中查看并上传文件了。
这个过程可能因操作系统和软件版本的不同而略有差异，如果你遇到任何问题，可以查阅相关的文档或指南进行参考。总的来说，按照上述步骤进行操作，你就能够轻松地配置和使用Webd自建网盘工具了
```
